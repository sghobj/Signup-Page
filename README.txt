To run this app you need node js installed at your device and tables in mysql to be created using the script file.

1. Use start-server.bat to start the server

2. go to localhost:3000

3. Signup up by entering you data

4. a submission of the form wll redirect you to a page where it tells you successful operation

5. Use the button in /success to go to the list of all database users who subscribed.

6. You can also use the url localhost:3000/user-list to check the list
